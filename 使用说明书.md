# Aim-DiyBot 使用说明书

Aim-DiyBot 是一款功能强大的游戏准心辅助工具，支持自定义样式、颜色、大小，并具备快捷键隐藏、开机自启等实用功能。

## ✨ 主要功能

*   **🎯 多样化样式**：支持十字、圆点、圆圈、混合模式，甚至可以使用**自定义图片**（支持 PNG/GIF 等）。
*   **⌨️ 快捷键控制**：支持绑定任意按键作为快捷键，游戏中一键隐藏/显示准心。
*   **🛠️ 自由调节**：可调节准心的大小、粗细、颜色、圆点大小。
*   **📍 精准定位**：
    *   **拖动模式**：按住按钮随意拖动准心位置。
    *   **微调模式**：使用方向键进行像素级微调。
    *   **一键居中**：快速回归屏幕中心。
*   **💾 方案管理**：支持保存多套配置方案，并可导出分享给好友。
*   **🛡️ 管理员模式**：内置管理员提权功能，确保在全屏游戏中稳定显示且快捷键生效。
*   **🚀 便捷体验**：支持开机自启、最小化到托盘。

## 📖 快速上手

### 1. 基础设置
打开软件后，在面板上方选择你喜欢的准心**类型**（如十字、圆点），点击**颜色**条选择颜色，拖动滑块调整**大小**和**粗细**。

### 2. 位置调整
*   **粗调**：点击并按住“**按住拖动准心**”按钮，将准心拖到大致位置。
*   **微调**：点击 X 或 Y 输入框，然后使用键盘的 **↑ ↓ ← →** 键进行精确移动。
*   **复位**：点击“**居中**”按钮，准心立即回到屏幕正中央。

### 3. 快捷键设置（新功能）
*   **绑定**：点击底部的“**绑定隐藏准星键**”按钮，然后按下你想用的键（例如 `F1` 或 `H`）。
*   **使用**：绑定后，按一下该键即可隐藏准心，再按一下即可显示。
*   **解绑**：点击“绑定”按钮后，按下 **ESC** 键即可清除快捷键。

### 4. 保存与分享
*   **保存**：调整好满意的参数后，在“方案”下拉框输入一个名字，点击“**保存**”。
*   **分享**：点击“**分享**”按钮，生成 `.json` 文件发给朋友。
*   **导入**：点击“**导入**”按钮，加载朋友分享的配置。

## ⚠️ 常见问题与解决方案

### Q1: 游戏中按快捷键没反应？
**A:** 请务必点击软件底部的“**管理员启动**”按钮。
*   原因：很多游戏（如 CS2、Apex）以管理员权限运行，如果准心工具权限比游戏低，Windows 会禁止它拦截按键。

### Q2: 全屏游戏中看不到准心？
**A:** 
1.  优先尝试点击“**管理员启动**”。
2.  如果仍不显示，请将游戏显示模式改为“**无边框窗口化 (Borderless Windowed)**”。
    *   *原理：独占全屏模式下，游戏画面会覆盖所有 Windows 窗口。*

### Q3: 如何彻底退出？
**A:** 点击右上角关闭按钮会最小化到系统托盘（右下角小图标）。如需彻底退出，请在托盘图标上右键选择“退出程序”。
